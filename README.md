# Fintra - Personal Finance Tracker

A modern personal finance tracking application with Brankas integration for automatic transaction sync from Philippine e-wallets and banks.

## Features

- Track income and expenses
- Manage multiple accounts (cash, cards, bank accounts)
- Connect e-wallets (GCash, Maya, ShopeePay) and banks via Brankas
- Automatic transaction import from connected accounts
- Budget management
- Debt tracking
- Analytics and insights
- Dark/light mode support

## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Development

The app runs on:
- Local: http://localhost:5173/
- Network: http://192.168.x.x:5173/ (accessible from other devices on WiFi)
